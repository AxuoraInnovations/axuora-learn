# Axuora Learn

A Next.js version of your Framer design — same structure, content, and styling, editable in Cursor.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## What’s included

- **Home** – Hero, Who are We, Features, Pricing, Benefits, Integrations, Blog, FAQ, CTA
- **Pages** – About, Pricing, Contact, Features, Integrations, Career, Waitlist, Blog list & post
- **Layout** – Navigation and Footer (matches Framer)
- **Design** – Archivo font, Framer colors (primary blue, theme palette), Tailwind

## Project structure

- `src/app/` – App Router pages
- `src/components/` – Navigation, Footer, and section components
- `src/components/sections/` – Hero, Features, Pricing, Benefits, FAQ, CTA, etc.
- `tailwind.config.ts` – Theme colors aligned with Framer

You can edit any page or component in Cursor and refresh to see changes.
